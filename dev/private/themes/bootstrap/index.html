lang

封装语言增强及一些工具方法。

lang.type(obj)

返回 obj 的类型。

返回值

  • boolean
  • number
  • string
  • function
  • array
  • date
  • regex
  • object
  • arguments
  • null
  • undefined

lang.each(obj, iterator, context)

遍历数组或者对象。可以绑定上下文。如果遍历数组，则迭代函数的参数为(element, index, list)。如果遍历对象，则迭代函数的参数为(value, key, list)。委派给原生forEach执行。

each([1, 2, 3], function(num){ alert(num); });
=> 按顺序alert每一个数字...
each({one : 1, two : 2, three : 3}, function(num, key){ alert(num); });
=> 按顺序alert每一个数字...
