游戏内容

  1. 游戏提供多种种族供玩家选择，但是在各种游戏模式下各有所限制。
  2. 如果提供种族选择，玩家必须在游戏开始前选定种族，游戏过程中不允许更换种族。
  3. 游戏过程中，击杀对方的兵将有可能随机掉落物品或金币。
  4. 玩家可以通过商店购买更多的附加武器、装备、技能、道具，以便打造更加强大的战斗群。
